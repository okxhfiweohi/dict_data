## exchange forms

| 类型 | 说明 |
| ---- | ----- |
| p | 过去式（did）|
| d | 过去分词（done） |
| i | 现在分词（doing） |
| 3 | 第三人称单数（does） |
| r | 形容词比较级（-er） |
| t | 形容词最高级（-est） |
| s | 名词复数形式 |
| b | Lemma, 基本形式 |
| f | Lemma 的变换形式 |

> [!NOTE]
>
> b: (basic)如 perceived 的 Lemma 是 perceive
>
> f: (form) Lemma 的变换形式，比如 s 代表 apples 是其 lemma 的复数形式
